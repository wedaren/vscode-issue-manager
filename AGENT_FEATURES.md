# 🚀 三个令人惊叹的 LLM + Agent 超能力功能

## 概述

基于对「问题管理器」项目的深入理解，我创建了三个将 LLM 与 Agent 能力完美结合的功能。这些功能不仅能帮助用户管理知识，更能**主动发现知识价值**、**指导学习路径**、**激发创新灵感**。

---

## 🧠 功能一：知识连接 Agent (Knowledge Graph Agent)

### 核心理念
> "知识的价值不仅在于单点，更在于连接"

### 令人惊叹之处
- **发现隐藏关联**：Agent 会深度分析你知识库中每个问题的语义内容，发现那些你可能从未意识到的隐藏关联
- **多种关系类型**：语义相似、概念重叠、因果关系、前置知识、扩展延伸、矛盾对立等
- **知识孤岛识别**：自动发现那些与主知识网络脱节的"孤岛"知识点，并提供整合建议
- **置信度评估**：每个发现的连接都有 AI 评估的置信度分数

### 使用方式
```
@issueManager /知识图谱
@issueManager /连接
```

### 技术亮点
- 批量语义提取：使用 LLM 从每个知识点提取关键词、核心概念、所属领域
- 智能聚类：自动将知识点分组，发现主题聚类
- 候选对筛选：使用概念重叠度和领域相似度预筛选有潜力的连接对
- 深度验证：使用 LLM 验证和解释每个连接的合理性

## 🎯 功能二：学习路径 Agent (Learning Path Agent)

### 核心理念
> "学习不应该是盲目的，而应该有路径、有目标、有检验"

### 令人惊叹之处
- **理解学习目标**：Agent 首先理解你真正想学什么，并细化成可衡量的目标
- **依赖分析**：自动分析知识点之间的前置依赖关系
- **拓扑排序**：使用算法生成最优的学习顺序
- **阶段划分**：将学习路径划分为清晰的阶段，每个阶段有里程碑
- **检验问题**：为每个阶段生成检验问题，确保真正掌握

### 使用方式
```
@issueManager /学习路径 掌握 TypeScript
@issueManager /学习 系统设计
@issueManager /path 机器学习基础
```

### 输出示例
- 📊 学习概览（总知识点、总时长、难度分布）
- 🗺️ 分阶段学习计划（入门→核心→进阶→实战）
- ⏱️ 每个知识点的预计学习时间
- ✅ 每个阶段的检验问题
- 🎓 最终学习成果清单

## 💡 功能三：创意激发 Agent (Idea Spark Agent)

### 核心理念
> "伟大的创新往往来自不同领域知识的碰撞" —— Arthur Koestler 的 Bisociation 理论

### 令人惊叹之处
- **跨领域碰撞**：Agent 会从你知识库的不同领域中随机选择概念，强制它们碰撞
- **多种创意方法**：
  - 🔀 **类比迁移**：将A领域的模式应用到B领域
  - 🎯 **组合融合**：将两个概念合二为一
  - ⚔️ **对比启发**：从差异中发现新视角
  - 🌟 **抽象提升**：找到共同的底层原理
  - 🔄 **逆向思考**：反转假设产生新想法
- **创意评估**：每个创意都有新颖度、可行性、影响力三个维度的评分
- **一键保存**：精彩的创意可以一键保存为问题文档

### 使用方式
```
@issueManager /创意              # 随机激发
@issueManager /灵感 AI产品       # 聚焦主题
@issueManager /激发              # 开始碰撞
@issueManager /spark            # 英文命令
```

### 输出示例
```
💡 创意: 知识版本控制系统

碰撞来源: "Git 分支管理" (技术) × "知识进化论" (哲学)

🎯 核心洞见: 
知识不是静态的，而是像代码一样在不断演进。
我们可以用 Git 的分支、合并、冲突解决等概念来管理知识的版本和演进。

评分: 🆕 新颖度 85 | ⚙️ 可行性 70 | 🎯 影响力 80

[💾 保存为问题]
```

## 技术架构

### Agent 设计模式
所有三个 Agent 都采用了 **ReAct (Reasoning + Acting)** 模式：

1. **Thought（思考）**：Agent 记录自己的推理过程
2. **Action（行动）**：基于思考执行具体操作
3. **Observation（观察）**：获取行动结果
4. **Loop（循环）**：根据观察调整下一步思考

### 共同特点
- 🔄 **多轮迭代**：不是一次性生成，而是多轮探索
- 💭 **思考可见**：Agent 的思考过程对用户透明
- ⏹️ **可取消**：支持随时取消长时间运行的任务
- 📊 **进度反馈**：实时显示分析进度
- 💾 **结果持久化**：支持保存分析结果
