---
# 扩展章节 Prompt 模板
---

请为指定章节标题输出该节的 Markdown 内容（不要包含章节编号；可使用二级或三级标题）。

- 文档标题：{{title}}
- 章节标题：{{heading}}
- 上下文：
```
{{selection}}
```

如果提供了本地参考文档（sources），请在适当处引用来源并在本节末尾标注所使用的参考条目编号：
```
参考文档：
{{sources}}
```

生成要求：
- 输出段落、子标题与要点（3-8 段为宜）
- 尽量使用中文专业表达，避免主观臆断
- 如果使用了参考文档内容，请在段落处标注引用（例如脚注或方括号引用）

只返回该章节的 Markdown 内容，不要包含额外解释。
