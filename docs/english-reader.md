# 英文阅读器（English Reader）使用说明

概述：英文阅读器为扩展提供一个可抓取英文网页并借助 LLM 进行翻译、摘要和交互问答的初始实现。

快速上手：
- 命令面板中运行 “打开英文阅读器” 或使用命令 `issueManager.openEnglishReader`。
- 在弹出的面板中输入目标文章 URL，点击“加载”。
- 文章加载后会分段显示，每段可快速填充到提问框进行翻译/解释。

注意：当前实现为原型：
- 抓取使用全局 `fetch`（在 Node 环境没有时将尝试动态导入 `node-fetch`）。
- LLM 调用为占位实现，需要替换为真实的 API 调用并增加 API key 管理。
