# 🎉 问题管理插件 v0.1.0 发布说明 🎉

我们非常高兴地宣布，**问题管理插件 (Issue Manager)** 迎来了首个重要里程碑版本 `v0.1.0`！

经过多个迭代版本的开发与优化，此版本汇集了从 `v0.0.1` 到 `v0.0.27` 的全部心血，旨在为 VS Code 用户提供一个前所未有的、集智能创建、多维组织和高效追踪于一体的本地化问题管理解决方案。

以下是 `v0.1.0` 版本包含的核心功能亮点：

## ✨ 核心功能

### 1. 🧠 **智能新建问题 (Intelligent Issue Creation)**
- **LLM 驱动**: 只需一个模糊的想法，AI 即可为您优化标题、并自动检索知识库中的相关笔记，避免重复劳动。
- **统一选择界面**: 在一个 QuickPick 界面中同时完成“创建新笔记”或“打开已有笔记”的操作，支持多选批量处理。
- **聊天内容一键记录**: 与 VS Code Copilot 深度集成，在聊天中通过自然语言（如“创建笔记”）即可将重要讨论一键保存为结构化的 Markdown 笔记。

### 2. 🌳 **多维问题视图 (Multi-dimensional Issue Views)**
- **问题总览 (Issue Overview)**: 以树状结构可视化您的整个知识体系，支持拖拽排序、多重引用，并可通过“移动到...”命令轻松重构。
- **孤立问题 (Isolated Issues)**: 您的“收件箱”，自动收集所有新创建但尚未归类的笔记，确保信息不遗漏。
- **关注问题 (Focused Issues)**: 您的“待办清单”，将重要问题固定在此处进行追踪。支持置顶，并以星标区分优先级。
- **最近问题 (Recent Issues)**: 您的“时光机”，自动记录您最近修改或创建的所有问题，支持按列表或按时间（今天、本周等）分组查看。
- **相关联问题 (Related Issues)**: 您的“上下文雷达”，点击任意问题，即可动态展示其在知识库中所有的关联位置，打破信息孤岛。

### 3. 🔗 **强大的关联与导航 (Powerful Linking & Navigation)**
- **拖拽即关联**: 支持从任意视图、编辑器标签页拖拽 `.md` 文件，轻松构建问题层级。
- **拖拽生成链接**: 从视图中拖拽问题到编辑器，自动生成标准的 Markdown 链接。
- **双向联动**: 在视图中点击问题，编辑器自动打开；在编辑器中打开问题文件，可右键查看相关联问题。
- **持久化状态**: 视图中节点的展开/折叠状态会被自动保存，保持您的工作区原样。

### 4. 📦 **自包含的数据管理 (Self-contained Data Management)**
- **本地优先**: 所有笔记都是您本地的 Markdown 文件，数据完全由您掌控。
- **便携的知识库**: 插件的所有元数据（结构、关注列表等）都存储在您指定目录下的 `.issueManager` 文件夹中，整个知识库可以轻松备份、迁移或通过 Git 进行版本控制。
- **无缝启动**: 插件启动时会自动检查配置，并提供清晰的引导，帮助新用户快速上手。
